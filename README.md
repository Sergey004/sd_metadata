# Metadata Editor

A web application for viewing and editing metadata from images generated by AI models (Stable Diffusion and others).

## Features

### View Metadata
- Drag-and-drop loading of PNG and JPEG images
- Extraction and display of prompt, negative prompt, and generation parameters
- EXIF and XMP data support
- Copy information to clipboard

### Edit Metadata
- Edit prompt and negative prompt
- Modify generation parameters
- Prepare to save edited images

### View LoRA Models
- Load .safetensors files
- Display model information: base model, VAE, training parameters
- Training statistics: epochs, steps, time
- Tag frequency and suggested prompt

## Supported Formats

- Images: PNG, JPEG, JPG
- Models: .safetensors

## Project Structure

- `index.html` — main HTML structure
- `script.js` — application logic, file handling, metadata extraction
- `style.css` — interface styling

## Technologies

- Pure JavaScript (ES6+)
- CSS Custom Properties for theming
- Font Awesome for icons
- piexifjs for JPEG EXIF data
